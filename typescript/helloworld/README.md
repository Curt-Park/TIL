## Install TypeScript vis npm
```bash
$ brew install npm
$ npm install -g typescript
$ npm install -g ts-node
```

## Compile (.ts -> .js)
```bash
$ npx tsc app.ts  # creates app.js
```

## Run with `node`
```bash
$ node app.js
Hello, World!
```

## Run with `ts-node`
```bash
$ ts-node app.ts
Hello, World!
```
