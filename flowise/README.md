# flowise quickstart

- repo: https://github.com/FlowiseAI/Flowise
- docs: https://docs.flowiseai.com/

## Setup
```bash
docker run --name flowise -d -p 3000:3000 flowiseai/flowise:latest
```

open http://localhost:3000/
